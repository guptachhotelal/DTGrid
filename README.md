## Getting Started

* Clone the latest code [here](https://github.com/guptachhotelal/DTGrid),
* Import the code in Eclispe/ Spring tool Suite
* Run or Debug the solution and you'll have 10000000(configurable) rows being server-side processed with DataTable plugin
